Descrição

Você quer aprender sobre desenvolvimento web? Então você está na página certa! Nesse curso daremos início ao aprendizado de HTML5 que é uma das linguagens que utilizamos para desenvolver websites.

HTML significa Hypertext Markup Language que significa Linguagem de Marcação de Hipertexto. HTML é a linguagem base da Internet e é preciso sabê-la para desenvolver websites. HTML é uma linguagem baseada em marcação, nós marcamos os elementos para mostrar quais informações a página exibe!

Nesse curso você irá aprender HTML de forma prática, dinâmica e didática. Todo o curso é em vídeo-aulas, você poderá acessar quantas vezes quiser e onde quiser, você terá acesso vitalício! Você aprenderá com Marcos Castro que possui mais de 7 mil alunos espalhados por mais de 65 países, é garantia de satisfação!
